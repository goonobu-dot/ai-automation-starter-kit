# Recovery Plan / 復旧計画

## English

If a case is routed incorrectly in this local sample:

1. Stop further intake into the proof-lab workspace.
2. Move the affected case to the quarantine review list.
3. Record the case id, reason, and reviewer note in the local audit trail.
4. Re-check the source invoice and order export.
5. Re-run the case only after a human reviewer confirms the correction.

This plan is local and reversible. No external action is triggered from this sample.

## 日本語

このローカルサンプルで誤った経路判定が起きた場合:

1. Proof Lab への追加取込を止めます。
2. 該当ケースを隔離レビュー一覧へ戻します。
3. ケースID、理由、レビューメモをローカル監査記録へ残します。
4. 元の請求書と注文書エクスポートを再確認します。
5. 人が修正を確認した後だけ再実行します。

この計画はローカルで完結し、元に戻せます。外部アクションは発火しません。
