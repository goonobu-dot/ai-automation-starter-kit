Business Automation Plan

The demo focuses on decision artifacts a team can inspect, repeat, and hand off.

What the plan gives you

  1. Recommended repositories worth reviewing first.
  2. License and maintenance risk before reuse.
  3. Which starter-kit template to use next.
  4. Dry-run implementation path before production use.
  5. Success metrics such as Manual handoffs reduced and time saved per repeated workflow.

Run This Demo

No external services required for local examples. GitHub discovery uses public GitHub data; set GITHUB_TOKEN only when you need higher API limits.

python3 -m venv .venv source .venv/bin/activate python3 -m pip install --upgrade pip setuptools python3 -m pip install -e . ai-automation-kit github-discover --business-area operations --limit 2 --output .tmp/operations-discovery sed -n '1,180p' .tmp/operations-discovery/business_automation_plan.md
