# Use Cases

This page explains where AI Automation Starter Kit can help with business automation.

## Operations

Use it for workflow orchestration, handoff reduction, approval routing, status reporting, and task tracking automation.

```bash
ai-automation-kit github-discover --business-area operations --limit 5 --output .tmp/operations-discovery
```

## Support

Use it for ticket triage, helpdesk automation, FAQ support, chatbot review, and escalation routing.

```bash

```

## HR

Use it for onboarding, recruiting support, resume review assistance, employee FAQ support, and internal request routing.

```bash
ai-automation-kit github-discover --business-area hr --limit 5 --output .tmp/hr-discovery
```

Each run can produce candidate rankings, an executive decision brief, a pilot scorecard, a value plan, a rollout map, a risk register, an audit plan, and an adapter starter when a safe prototype candidate exists.
