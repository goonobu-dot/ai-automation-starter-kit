# ユースケース

このページでは、AI Automation Starter Kit がどのような業務自動化に役立つかを説明します。

## Operations

ワークフロー自動化、作業の受け渡し削減、承認整理、定期報告、タスク管理に使えます。

```bash
ai-automation-kit github-discover --business-area operations --limit 5 --output .tmp/operations-discovery
```

## Support

チケット分類、ヘルプデスク自動化、FAQ対応、チャットボット検討、エスカレーション整理に使えます。

```bash
ai-automation-kit github-discover --business-area support --limit 5 --output .tmp/support-discovery
```

## Finance

請求書処理、経費確認、スプレッドシート整理、レポート作成、承認状況の管理に使えます。

```bash
ai-automation-kit github-discover --business-area finance --limit 5 --output .tmp/finance-discovery
```

## Sales

リード情報整理、CRMワークフロー自動化、フォローアップ下書き、パイプライン確認、アカウント調査に使えます。

```bash
ai-automation-kit github-discover --business-area sales --limit 5 --output .tmp/sales-discovery
```

## Marketing

コンテンツ作成、キャンペーン計画、SEOチェック、SNS投稿準備、マーケティングレポート作成に使えます。

```bash
ai-automation-kit github-discover --business-area marketing --limit 5 --output .tmp/marketing-discovery
```

## HR

入社手続き、採用業務の補助、履歴書確認、社員向けFAQ対応、社内依頼の振り分けに使えます。

```bash
ai-automation-kit github-discover --business-area hr --limit 5 --output .tmp/hr-discovery
```

各実行では、候補ランキング、導入判断資料、効果測定用の表、価値計画、関係者整理、リスク一覧、監査計画、安全に試せる候補がある場合の adapter starter を作れます。
