# フロー選択ガイド

AI Automation Starter Kit には 64 件の業務フローがあります。最初の案件では、難しいAIエージェントより、企業がすぐ理解できるフローを選びます。

## おすすめの選び方

```bash
ai-automation-kit flow-guide --industry finance --niche accounting --output .tmp/flow-guide
```

`recommended_flows.md` を開き、上位候補を確認します。

## 初心者に向いているフロー

- 請求書・書類の回収
- 問い合わせ返信の下書き
- KPIレポート作成
- 承認依頼のルーティング
- CRMリード整理
- 日程調整やリマインド

## 避けた方がよい初回案件

- 決済や送金を自動実行するもの
- 採用・解雇・与信などの判断を自動化するもの
- 医療、法律、税務の判断を含むもの
- 顧客の本番データへ直接書き込むもの

## 判断基準

良い初回フローには、次の特徴があります。

- 入力データが明確
- 出力が下書きやレポートで済む
- 人間の承認者がいる
- 月間件数を数えられる
- 効果を時間削減や遅延削減で測れる
